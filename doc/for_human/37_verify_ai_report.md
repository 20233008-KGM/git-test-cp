# AI 리포트 집계 확인 — 3분

> **전제:** [29](./29_supabase_bundle_sql.md) 번들 v2 완료 · 앱 `npm run dev`  
> **관련:** [30](./30_edge_ai_report.md) (LLM 문단은 H-002)

코딩 없이 **DB 집계**만 확인합니다. Edge·OpenAI 없어도 됩니다.

---

## 1. 학생 — DB 미리보기

| # | 동작 | 기대 |
|---|------|------|
| 1 | 로그인 → **마이페이지** | 페이지 로드 |
| 2 | **DB 활동 미리보기 (A4)** 클릭 | A4 제목 표시 |
| 3 | 파란 **집계:** 줄 확인 | `트러블슈팅` · `산출물` · `피드백` · `회고` · `동료평가` · **`교수평가 N/M팀`** |

- 번들 v2 **전**: 피드백·회고·동료·교수평가는 대부분 **0**
- 팀 활동(트러블슈팅·산출물) 후 숫자 증가

---

## 2. 교수 평가 반영 (선택, +2분)

| # | 동작 | 기대 |
|---|------|------|
| 1 | **교수** 계정 → 팀 입장 → **학생 평가**·**프로젝트 평가** 저장 | 에러 없음 |
| 2 | **해당 학생** 마이페이지 → DB 미리보기 | `교수평가 1/1팀` 등 (학생평가·프로젝트평가 각각 반영) |
| 3 | A4 초안 **팀 섹션** | **팀 피드백** · **동료평가** · **회고 요약** · **교수 피드백** (각 저장 후) |
| 4 | A4 **성장 회고** 문단 | 회고·교수 평가 등 스니펫이 문단에 반영 (LLM 전 DB 초안) |
